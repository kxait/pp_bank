# pp_bank
projekt banku na zajęcia z podstaw programowania

2 semestr informatyka w inżynierii komputerowej WIEIK PK

## funkcjonalności:
- konta
- lista transakcji
- ładowanie z i do pliku CSV
- UI napisane w qt
- liczenie balansu kont z listy transakcji
- wypisywanie raportów
